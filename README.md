This project is a modern and interactive implementation of the classic Rock, Paper, Scissors game built using HTML, CSS, and JavaScript.

The game allows users to play against a computer opponent in real-time with dynamic UI updates and instant result feedback. It demonstrates core web development concepts such as DOM manipulation, event handling, and responsive design.

Rock Paper Scissors is a simple hand game where each player chooses one of three options — rock, paper, or scissors — and the winner is decided based on predefined rules .

🚀 Features
🎯 Play against computer (AI-based random selection)
⚡ Instant result display (Win / Lose / Draw)
📊 Score tracking system
🎨 Interactive and responsive UI
💻 Works across desktop and mobile devices
🔁 Real-time game updates without page reload
🛠️ Tech Stack
HTML5 – Structure of the game
CSS3 – Styling and layout
JavaScript (Vanilla JS) – Game logic and interactivity
🎮 How to Play
Select one option: Rock, Paper, or Scissors
The computer randomly selects its move
Result is decided based on rules:
Rock beats Scissors
Paper beats Rock
Scissors beat Paper
Score updates automatically after each round
